# BJH AI Pro 正式測試版 V1.0

本版停止新增功能，集中完成整合、修正與核心測試。

## 已修正

- 快速啟動精靈引用錯誤的策略資料表
- 資料新鮮度中心引用錯誤的策略資料表
- 備份完成判斷引用不存在的稽核欄位
- 備份下載現在會留下稽核紀錄
- 加入全專案 JavaScript、匯入路徑、HTML ID、SQL 資料表與路由掛載檢查
- 加入持股平均成本、部分賣出、賣超防護與交易輸入測試

## 啟動方式

電腦需先安裝 Docker Desktop，然後在本資料夾執行：

```bash
docker compose up
```

瀏覽器開啟：

```text
http://localhost:3000
```

## 手機實際操作

iPhone／iPad 直接操作需要部署到 HTTPS 雲端主機。這個 ZIP 是可部署測試版程式，不包含公開測試網址。
