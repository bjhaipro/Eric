# BJH AI Pro Cloud Sprint 9

## 本版完成

- 每日資產快照
- 同一天重複儲存時自動更新
- 7／30／90／180／365 天走勢篩選
- 資產市值折線圖
- 期間市值變化與變化率
- 最近 10 筆快照明細
- iPhone／iPad 響應式圖表
- 快照資料依使用者帳號隔離

## API

- `GET /api/v1/snapshots?days=30`
- `POST /api/v1/snapshots/today`
- `DELETE /api/v1/snapshots`

## 使用方式

登入後，在「每日資產走勢」區按下「儲存今日快照」。
系統會依目前交易紀錄與手動股價，保存當日總成本、市值、未實現損益、已實現損益及報酬率。

目前股價仍是使用者手動輸入，不會假裝成即時行情。
